# StudentPerformanceAnalysis
Project title
Student Performance Factor Analysis
Objective
To analyse factors affecting students performance and predict exam scores
Tools Used
•	Python
•	Pandas
•	numpy
•	Ploty Express
•	Matplotlib
•	Scikit-learn
•	category_encoders
Dataset 
Dataset comprised of 6,402 samples with 18 features. it made of up 13 categorical features and 5 numerical features.
Process taken
1)	Preparation of data
•	import of data
•	Data cleaning
•	Exploratory analysis
•	Split data
2)	Build of model
•	Baseline
•	Iterate
•	Evaluate
•	Feature engineering
3)	Communicate the result.
Key insights
•	Study hours and attendance strongly influences exams scores.
•	Attendance is highly predictive.
•	Internet access has a less impact.
Model 
•	I used RandomForestRegressor 
•	I used SimpleImputer transformer
